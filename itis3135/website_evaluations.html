Website Evaluations

Instagram

Instagram is a social media platform that allows users to view and upload both videos and pictures to their accounts. Since its release in 2019, Instagram has become one of the most popular social media platforms.

  • Contrast: While Instagram uses colors that developers tend to avoid, black and white are a great contrast for one another. Content is easy to read and dark mode especially is easy on the eyes
  • Repetition: Instagram does especially well when it comes to repetition. Each post looks relatively the same besides the content itself. Account username is in the same place along with all the buttons the user can interact with.
  • Alignment: Instagram does suffer from a whitespace problem on the desktop version, but they do take measures to lessen the problem. When you first load the page there is whitespace between the left sidebar and the main content of the page, and the right sidebar and the main content of the page. Once you scroll down the entire right hand side of the page is whitespace. The page is generally neatly aligned
  • Proximity: Related buttons are grouped together. Similar elements are always found near each other.

Steakfry

Steakfry is the personal website of an artist with the same name. His discography is shown on the website, along with options to buy merchandise.

  • Contrast: The contrast on this website does not look good. The background is an ugly dark red and the text is black.
  • Repetition: This component is lacking although there is not much content to repeat elements. Some elements are repeated on the page when this is not needed.
  • Alignment: Steakfry's website has poor alignment. The content goes off the screen horizontally requiring the user to scroll both vertically and sideways. The spacing is also inconsistent between elements in the same section.
  • Proximity: The biggest problem with proximity on this page is that links to external websites, which should be grouped together, are on opposite sides of the page.
